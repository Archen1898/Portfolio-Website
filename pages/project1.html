Waste Recognition System

 The Waste Recognition System is an automated recycling bin that is capable of separating bottles from regular waste.

My Role on the Team

 I was responsible for designing and implementing the image recognition functionality of the project using a Raspberry Pi unit. I utilized an open source image recognition software called OpenCV and installed it via the command line onto the Raspberry Pi. Once I got the basic object recognition functionality up and running I was able to redesign the code to suite our project's needs. Once completed, I integrated the motor controls with the image recognition software then refactored the code to decrease the system's response delay by about 40%.

Demonstration

 This is a demonstration video created by my senior design team. In the video, my team member, Alejandro Aloma demonstrates the functionality of our Waste Recognition System. The System utilizes image recognition to analyse captured frames from a camera inside the System. From there, it compares each object inserted into the trash chute and determines whether or not it is a bottle. If it is a bottle, it moves the object to the recycling bin and if not, it sends it to the regular trash bin. Each bin also has an ultrasonic sensor attached to determine its fullness. Once the height of the trash within each bin reaches a predetermined level, the LED light lights up to indicated that the bin is full.
